# Verde Heights - Luxury Eco Resort Website

Verde Heights is a modern luxury resort website designed to provide an elegant and immersive hospitality experience. The project features premium room listings, an interactive booking section, dining services, wellness facilities, special offers, and guest testimonials.

## Features

- Responsive Design
- Luxury Hotel Landing Page
- Interactive Booking Form
- Room & Suite Showcase
- Dining & Wellness Sections
- Guest Testimonials Slider
- Special Offers Section
- Mobile-Friendly Navigation
- Smooth Animations & Effects

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Project Type

Hospitality / Hotel Website
